# 理财知识网站 - 产品需求文档 (PRD)

## 一、项目概述

### 1.1 项目名称
理财知识网站（暂定）

### 1.2 项目定位
基于 Next.js + shadcn/ui 的现代化理财知识平台，接入 DIFY Agent 提供智能问答与个性化理财建议，帮助用户系统学习理财知识、建立正确投资观念。

### 1.3 技术栈
- **前端**：Next.js 15、React 19、Tailwind CSS、shadcn/ui
- **AI 能力**：DIFY Agent（大模型驱动的智能助手）
- **部署**：待定

---

## 二、核心功能建议

### 2.1 智能理财助手（DIFY Agent 核心场景）

| 功能 | 描述 | 优先级 |
|------|------|--------|
| **智能问答** | 用户输入理财相关问题，Agent 基于知识库回答（基金、股票、保险、储蓄、负债管理等） | P0 |
| **理财规划咨询** | 根据用户年龄、收入、风险偏好、目标，给出资产配置建议 | P0 |
| **概念解释** | 解释专业术语（如复利、定投、ETF、年化收益率等） | P0 |
| **产品对比** | 对比不同理财产品（如基金 vs 股票、年金险 vs 增额寿等） | P1 |
| **风险测评** | 通过对话式问卷评估用户风险承受能力 | P1 |

### 2.2 知识库与内容

| 功能 | 描述 | 优先级 |
|------|------|--------|
| **理财百科** | 结构化理财知识文章（分类：入门、进阶、投资、保险、税务等） | P0 |
| **热门话题** | 定期更新的理财热点解读（如降息、新政策、市场动态） | P1 |
| **案例学习** | 真实/模拟理财案例，供用户学习与讨论 | P2 |
| **工具与计算器** | 复利计算器、定投收益模拟、退休规划计算器等 | P1 |

### 2.3 用户与互动

| 功能 | 描述 | 优先级 |
|------|------|--------|
| **对话历史** | 保存用户与 Agent 的对话记录，支持回顾与继续 | P0 |
| **收藏与笔记** | 收藏文章、保存 Agent 回答要点 | P1 |
| **学习路径** | 推荐从入门到进阶的学习顺序 | P2 |
| **每日一题/小贴士** | 轻量级每日学习提醒 | P2 |

### 2.4 个性化与运营

| 功能 | 描述 | 优先级 |
|------|------|--------|
| **用户画像** | 根据对话与行为，推断用户阶段（小白/进阶/专业） | P2 |
| **个性化推荐** | 根据画像推荐适合的知识与工具 | P2 |
| **订阅与通知** | 新内容、新功能订阅（可选邮件/站内） | P2 |

## 三、DIFY Agent 集成方案

### 3.1 集成方式
- 通过 DIFY API 调用 Agent
- 支持流式输出（Streaming）提升体验
- 支持多轮对话上下文传递

### 3.2 Agent 能力规划
- **知识库**：上传理财领域文档、FAQ，作为 Agent 检索增强（RAG）数据源
- **工作流**：可配置「问题分类 → 检索 → 生成」等流程
- **工具调用**：可扩展调用计算器、外部 API（如基金净值查询）等

### 3.3 安全与合规
- 明确声明「仅供参考，不构成投资建议」
- 敏感话题（如具体个股推荐）需有免责与风险提示
- 用户数据脱敏与隐私保护

## 四、页面与信息架构建议

```
首页
├── 智能助手入口（突出展示）
├── 热门知识 / 推荐阅读
└── 工具入口（计算器等）

智能助手页
├── 对话区域
├── 快捷问题 / 示例问题
└── 对话历史侧边栏

知识库
├── 分类导航（入门 / 投资 / 保险 / 税务等）
├── 文章列表
└── 文章详情

工具中心
├── 复利计算器
├── 定投收益模拟
└── 退休规划计算器

个人中心（可选）
├── 对话历史
├── 收藏与笔记
└── 学习进度
```

## 五、MVP（最小可行产品）范围建议

**第一阶段（MVP）**：
1. 首页 + 智能助手对话页（接入 DIFY Agent）
2. 基础知识库（10–20 篇核心文章）
3. 对话历史（本地存储或简单后端）
4. 1–2 个实用工具（如复利计算器、定投模拟）

**第二阶段**：
- 完善知识库与分类
- 用户系统、收藏、笔记
- 更多工具与个性化推荐

**第三阶段**：
- 学习路径、每日一题
- 运营与数据分析
- 移动端优化

## 六、非功能需求

| 类型 | 要求 |
|------|------|
| **性能** | 首屏加载 < 3s，Agent 首字响应 < 2s |
| **兼容** | 支持 Chrome、Safari、Edge 等主流浏览器 |
| **无障碍** | 基础键盘导航与语义化 HTML |
| **SEO** | 知识库文章支持 SSR/SSG，利于搜索引擎收录 |
